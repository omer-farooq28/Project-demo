##  Sample Calculator (Using Lambda Functions)

A simple calculator application built using **lambda functions** to perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

* Uses lambda functions for operations
* Supports Addition, Subtraction, Multiplication, Division
* Clean and simple logic
* Beginner-friendly project

## Technologies Used

* Python

##  How It Works

The calculator uses lambda functions to define operations in a concise way.

Example:

```python
add = lambda a, b: a + b
```
